## Doctor
- As a doctor, I want to view my scheduled appointments so that I can manage my patients.
- As a doctor, I want to add prescriptions so that I can provide treatment.
- As a doctor, I want to update appointment status so that patients know their appointment results.

## Patient
- As a patient, I want to book appointments so that I can receive medical attention.
- As a patient, I want to cancel appointments so I can free up my schedule.
- As a patient, I want to view my prescriptions so I can follow my treatment.

## Admin
- As an admin, I want to manage all users so that the system remains organized.
- As an admin, I want to view all appointments so I can ensure operations are running smoothly.
- As an admin, I want to manage doctor schedules so I can coordinate resources.
